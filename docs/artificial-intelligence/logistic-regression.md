# Logistic Regression

· 로지스틱 회귀(Logistic Regression)는 이름은 회귀이지만 분류 모델입니다.

· 이 알고리즘은 선형 회귀와 동일하게 선형 방정식을 학습합니다.
 
· 선형 회귀는 예측 결과를 확률로 표현하지만 로지스틱 회귀는 0 혹은 1로 표현합니다.
